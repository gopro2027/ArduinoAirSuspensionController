Remix of Itchyshaft Waveshare 3.5 controller case. 
- Custom logo
- Slightly different tolerances 
- Changed USB C port
- Shortened button a smidge
- Meant for NO battery or a very small 3.7v cell under 50mm L x 12mm W x 6mm H. Unfortunately, most of them are quite low mAh, but it's an option.
- DXF file included for everyone's use. 

New Case is a Depth of 14.2mm, and it's probably already about as thin as it could get.
